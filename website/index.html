The Library of Congress has launched the Bibliographic Framework Initiative to better accommodate future needs of the library community. Zepheira and The Library of Congress are working together to develop a Linked Data model, vocabulary and enabling tools / services for supporting this Initiative.

MARCR.org provides an overview of this effort. This site offers:

  • Access to the demo area, where the model is being tested and displayed using different test suites of data
  • View of the MARCR vocabulary (along with formal specifications)
  • Educational resource on how MARCR vocabulary relates to MARC21 fields and subfields
  • A library of resources for learning about related initiatives in Linked Data and other serialization and cataloging projects.
  • Code and supporting servies.
  • Project timeline and supporting deliverables.

Additional background on this initiative can be found here:

  • The Library of Congress Announces Modeling Initiative (May 22, 2012)
  • A Bibliographic Framework for the Digital Age (October 31, 2011)
  • Transforming our Bibliographic Framework: A Statement from the Library of Congress (May 13, 2011)

MARCR.org is a joint effort of US Library of Congress and Zepheira
