The Library of Congress has launched the Bibliographic Framework Transition Initiative to better accommodate future needs of the library community. A major focus of the initiative will be to determine a transition path for the MARC 21 exchange format to more Web based, Linked Data standards. Zepheira and The Library of Congress are working together to develop a Linked Data model, vocabulary and enabling tools / services for supporting this Initiative.

WERCR (WEb Readable Cataloging Resources) provides an overview of this effort. This site offers:

  • A demonstration area where the model is being tested and displayed using different test suites of data
  • View of the WERC vocabulary along with supporting formal specifications
  • Educational resource on how the WERC vocabulary relates to MARC21 fields and subfields
  • A library of resources for learning about related Linked Data initiatives, serialization and cataloging projects.
  • Code and supporting servies.
  • Project timeline and supporting deliverables.

Additional background on this initiative can be found here:

  • The Library of Congress Announces Modeling Initiative (May 22, 2012)
  • Transforming our Bibliographic Framework: A Statement from the Library of Congress (May 13, 2011)
  • A Bibliographic Framework for the Digital Age (October 31, 2011)
