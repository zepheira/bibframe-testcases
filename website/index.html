The Library of Congress has launched the Bibliographic Framework Initiative to better accommodate future needs of the library community. A major focus of the project is to reflect the MARC 21 format as a Linked Data model while retaining as much as possible the robust and beneficial aspects of the historical format.

This site provides an overview of this effort, the MARC Linked Data model, supporting tools and resources to help support a transition path.
