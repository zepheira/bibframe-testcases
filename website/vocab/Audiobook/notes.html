Instance

Instance types are a combination of physical carrier listed under MARC Code List for Carrier Types and top level IANA media types for digital resources. Instance types are a plug point and specified by community defined application profiles.

Instance.actor

Refinement relationships (producer, publisher, distributor, manufacturer, etc.) are defined as subproperties of this property. As a start, I suggest we focus on a subset of the MARC relator terms what might be applicable to Instances. If we decide there is the need for more than a single entitiy associated with an Instance (an instance has more than one publisher / publishedAt / publishedOn combination of assertions) we'll need to materilize this pattern.
