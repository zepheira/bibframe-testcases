Abstract

As described in the document BIBFRAME Annotation Model there are many uses for annotations on the Web, whether comments on media, reviews of products, services, books, etc. The BIBFRAME Annotation Model describes a formalization of such annotations in terms of the Resource Description Framework (RDF) model.

There are already many lightweight applications of Annotations on the Web, implemented using bespoke models and synaxes. Zepheira is developing applications of BIBFRAME for seamless integration into such models, which requires a very lightweight, unobtrusive application of this model which is can be rapidly processed. This document describes an application of BIBFRAME wih the following characteristics:

  • Directly Embeddable within existing Web markup vocabularies, such as HTML5 and Atom
  • Easy to process with a high speed, efficient, streaming XML processor, including in Web scripting context
  • Trivially comprehensible to an average developer examining annotations using "view source"
